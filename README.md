# Online Store Management API

## Project Overview
This is a C# API developed as part of an Object-Oriented Programming (OOP) course.

The primary goal of this project was to implement core OOP principles.

## Key Features
* **Product Management:** Ability to add, view, and categorize products.
* **User Roles:** Implemented logic for both Customers.
* **Data Handling:** Use of classes and structures to maintain store data during runtime.
